```

  - Font: System bold or Playfair Display
  - Size: 24-28px mobile, 32-40px desktop
  - Line height: 1.2

  - Font: Georgia, Charter, or system serif
  - Size: 18-20px
  - Line height: 1.6
  - Max width: 680px

  - Font: System sans-serif
  - Size: 14px
```
